# Objetivo del proyecto

Crear una aplicación de venta de bienes raíces llamada Luxe Estate, usando NextJS (App Router) y Tailwind.

El proyecto ya está creado y con tailwind instalado, pero sin estilos ni funcionalidades. El objetivo es implementar el diseño y las funcionalidades básicas de una app inmobiliaria moderna, premium y minimalista, siguiendo exactamente los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta de "Antigravity" dentro del proyecto.
